## 自动化格式校正工具

可以在各个 IDE 的插件管理器里搜索类似“flake”、“pep8”、“lint”字样的插件，一般都是提供格式检查的功能。


### For Atom

Atom 用户可以安装 [atom-beautify](https://atom.io/packages/atom-beautify)，
一键提供多语言的自动格式化功能。

*（JSX 用户慎用该功能）*
